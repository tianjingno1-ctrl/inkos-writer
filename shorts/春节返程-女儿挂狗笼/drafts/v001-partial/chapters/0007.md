# 第7章 舆论的风暴

就在周建国还在苦苦哀求时，他的手机开始疯狂震动。他颤抖着接起电话，里面传出他公司总经理暴怒的吼声，声音大到在场所有人都能听到。

“周建国！你上热搜了你知道吗？‘狠心父亲将五岁女儿挂车外狗笼’，视频都传疯了！客户看到视频直接要求取消合作，还要我们赔偿品牌损失！你被开除了！公司会起诉你挪用公款，你等着坐牢吧！”

周建国瘫坐在地，手机滑落。

原来，周丽之前为了炫耀和抹黑我发的那些视频，被正义网友扒出了真相。

配上路人在服务区拍到的妞妞蜷缩在笼子里的照片，瞬间引爆了全网的怒火。

“这种人渣也配当父亲？”“建议死刑！把他也挂车外跑五百公里！”
“人肉他！他在哪家公司上班？”

网友的力量是惊人的，不到半小时，周建国的职位、周强的学校、甚至周老太住的村子都被扒了个精光。

周丽看着自己原本想用来涨粉的账号，此时全是排山倒海的谩骂，吓得尖叫一声，直接晕了过去。

周老太还在哭天抢地：“老天爷不长眼啊，儿媳妇害死全家啦……”

我哥林晓峰走过去，冷冷地拿出手铐。

“周建国，周强，你们涉嫌虐待罪和危险驾驶罪，现在正式逮捕你们。”

冰冷的手铐扣在周建国腕上的那一刻，他抬头看向我，眼神里全是绝望和怨毒。

我只是抱紧了怀里的妞妞，轻声说：“周建国，这只是个开始。”

医生把妞妞抬上担架，我跟着上了救护车。在车门关上的那一刻，我看到周建国被押上警车，那个他亲手焊的铁笼子，正孤零零地挂在车尾，像是一个讽刺的墓碑。

救护车呼啸而去，我握着妞妞冰凉的小手，眼泪终于掉了下来。

妞妞，妈妈带你回家。
