「为给白月光出气，男友群发我私密照」= **人物关系 + 极端作践 + 动机明确**

| 元素 | 读者预期 |
|------|----------|
| 白月光 | 有第三者，男主站队明确 |
| 群发私密照 | 隐私武器化、社死级伤害 |
| 出气 | 男主主动作恶，不是误会 |

读者点进来要的是：**作践有多狠，反杀就有多爽**——不是等他后悔的慢虐，是**亮底牌、动手、看他表情**。

和「追妻火葬场」的分工：
- **追妻**：心死离场 → 他事后追悔
- **本篇**：社死作践 → 女主主动反杀 → 当场社死/自食恶果

你的标题更偏后者。

## 怎么写：推荐走「现实社死复仇型」

按复仇打脸题材的标准骨架（8–11 节，约 1–1.5 万字）：

```
黄金简介（150–300字，结果倒序剧透反杀手段）
→ 第1章：社死现场慢放（照片流出/开盒/谩骂电话）
→ 门外偷听：撞破他和朋友真心话（动机一次性砸下）
→ 心死定格 + 反杀决意（亮底牌前瞻预告）
→ 逐章反击（回旋镖/物证/法律/身份揭晓）
→ 当众打脸 → 加害者自食恶果
```

### 1. 开篇：别铺垫，直接上炸弹

前 3 句必须是**事件/对话/信息炸弹**，不要先写天气、装修、走廊灯光。

可参考开篇模式（题材包里的真实范式）：

> 男友为了逗白月光开心，把我的私密照片传上了色情网站。当晚我被人开盒，谩骂和骚扰电话一夜没停。

或者更狠一点，冲突前置：

> 手机震了十七次，我才点开那条群消息——是我自己的照片，发件人备注：「我男朋友」。

### 2. 核心反转：提前想好「底牌」

私密照题材不能只有对骂，必须有**规则碾压**。常见三条路，选一条做主反转：

| 底牌类型 | 反杀方式 | 爽感来源 |
|----------|----------|----------|
| **职业身份** | 律师/市监局/网安/媒体从业者，走报案+平台下架+追责 | 专业碾压 |
| **隐藏资源** | 豪门千金/家族背景/投资人身份 | 身份反差 |
| **系统/穿书**（点众向） | 登机瞬间清除全网痕迹、弹幕审判 | 超自然爽 |
| **证据链** | 早有录音/聊天记录/转账记录，反手侵犯隐私罪 | 回旋镖 |

题材包里这篇范例的收尾是：**登机那一刻，所有相关视频和照片被系统清除**——偏点众/系统流。若走黑岩/七猫，更稳的是**报案 + 律师函 + 当众揭穿门外偷听那段话**。

### 3. 必写桥段：门外偷听

这是这类文的**头号导火索**。女主去找他对质，隔着门听见他和兄弟谈笑：

> 「现在所有人都把她床上的样子看光了，除了我，谁还敢接盘她？」

一句定生死，比写十段心理描写都管用。

### 4. 情绪曲线

```
社死痛（9）→ 偷听心死（10）→ 冷下来（决意反杀）→ 亮牌爽（10峰值）→ 漠视离场（余韵）
```

受虐段要够痛（心沉谷底、咬到血腥味），反杀段要够冷（不哭闹，一句审判定生死）。

### 5. 导语/简介公式（150–300 字）

黑岩/七猫通用模板：

> 正常生活 → 伤害事件 → 最残忍一句对话 → 女主反转暗示 → 悬念钩子

示例骨架：

> 恋爱三年，我以为他只是忘不了白月光。直到我的私密照出现在同学群里，我才在他别墅门外听见真相——他不过是用我，换她一笑。他们不知道的是，【你的底牌一句】。这一次，我不哭，也不走，我让他们亲口认栽。

### 6. 写作注意

- **第一人称「我」**，代入感最强
- 私密照/开盒写**后果**（电话、谩骂、不敢出门），不细化传播手法，避免平台敏感
- 反杀走**法律/规则/身份**，不写以暴制暴的具体伤害步骤
- 白月光用「娇弱软刀」：「姐姐，我从没怪过你，是他太心疼我」

## 适合发什么平台

| 平台 | 匹配度 | 理由 |
|------|--------|------|
| **黑岩** | ⭐⭐⭐⭐⭐ | 女频情绪密度最高，道德踩踏+极致刺激正是强项；私密照/开盒/门外偷听都适配 |
| **七猫** | ⭐⭐⭐⭐ | 「总裁豪门」「家庭复仇」爆款区，导语+第一人称+阶梯作践成熟 |
| **点众** | ⭐⭐⭐⭐ | 标题即剧情；若加系统/弹幕/读心，更贴平台味 |
| **知乎盐言** | ⭐⭐⭐ | 能发，但要降烈度、加专业细节（律师取证、报案流程），节奏更慢、后劲更大 |

### 我的建议

- **想最快起量、情绪拉满** → **黑岩** 或 **七猫**
- **标题已经够炸、想走情绪快餐** → **点众**（可保留系统清除类收尾）
- **想写成「有质感的世情」** → **知乎盐言**（女主设成律师/媒体人，反杀靠专业与证据链）

不太适合：番茄长篇连载（短篇一次性读完更合适）、小红书（篇幅和深度都不够，顶多做引流切片）。
