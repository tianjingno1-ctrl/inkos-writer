## 二、标题为什么能爆

点众公式：**场景 + 行为 + 后果/情绪**

```
春节返程          → 全民共鸣（高速、堵车、回婆家）
老公一家           → 集体施害，不是单人渣男
把我女儿           → 弱者/孩子钩子，护女本能拉满
挂车外狗笼子里     → 极端道德踩踏，3 秒引爆愤怒
```

**下沉读者代入点**

- 过年回婆家受气（不让坐前排、女儿被当外人）
- 远嫁后「你的娃不如他们的狗/侄子重要」
- 高速上被当免费保姆（抱行李、自己挤后排）

**狗笼** = 日常道具 + 极端羞辱，比抽象「虐待」更有画面、更易传播。

## 三、黄金简介（150–300 字，正文前）

```
春节返程，高速服务区，老公周建国把五岁的女儿妞妞塞进车尾外挂的狗笼里。
婆婆说：「妞妞占地方，狗笼透气，小孩子皮实。」
小叔子抱着他们的金毛坐进副驾，我抱着行李挤在后备箱。
女儿在笼子里喊妈妈，嗓子都哑了，周建国只说：「别矫情，就俩小时。」
他们不知道，我手机里存着行车记录仪全程录像。
服务区那个穿制服的，是我哥。
而周建国公司那份即将签的合同，甲方联系人，是我。
```

**导语必含 4 件事**

1. 狗笼事件（核心羞辱）
2. 夫家全员站队（集体作恶）
3. 母亲当时状态（忍 / 疯 / 冷——正文偏「先忍后冷」）
4. 反杀底牌预告（录像 / 人脉 / 法律）

## 九、12 章逐章方案（CLI 出纲 / 写稿依据）

### 第 1 章｜狗笼炸弹

- **主事件**：服务区，女儿被装进车尾外挂狗笼
- 婆婆让小叔子带狗坐副驾；周建国：「妞妞坐这个，透气」
- 女儿哭，婆婆：「哭两声就停，别惯着」
- 我冲上去被推开：「大庭广众的，别丢周家的脸」
- **章尾**：笼门扣上，小手从格栅伸出来喊妈妈
- **情绪**：震惊 8 → 愤怒 10

### 第 2 章｜高速上的三小时

- 我挤在后备箱，透后窗看狗笼
- 女儿声音越来越小；拍玻璃求停车被吼「别添乱」
- 闪念：结婚五年，四年过年没回娘家，女儿三次发烧被说装病
- 婆婆发朋友圈：返程快乐，配图金毛在副驾打盹
- 我确认行车记录仪在录 / 偷偷手机录像
- **章尾**：女儿喊不出声，我摸到她手指发紫
- **情绪**：愤怒 10

### 第 3 章｜服务区「休息」

- 他们先给狗喂水，不管女儿
- 我抱出女儿：嘴唇发紫、手冰凉
- 婆婆：「不就吹了点风，至于吗？」周建国：「真心疼，当初就别嫁我」
- 小姑子拍视频发家族群：「嫂子又作妖了」
- **章尾**：偷听周建国电话：「挂外面没事，就图个清净」
- **情绪**：愤怒 10 → 心死开关

### 第 4 章｜心死定格

- 心死句（独立成段）：「那一刻，我不是他的妻子，只是周家的免费保姆。」
- 翻手机：许律师未接来电、哥哥消息「到了说一声」
- 发现哥哥今日执勤该服务区；娘家担保周建国公司的事
- 拨通：「哥，他们把我女儿挂狗笼里了。」
- **章尾**：「报警。现在。」

### 第 5 章｜布局与围观

- 路人开始拍照；婆婆骂「关你什么事」
- 周建国抢手机要删视频
- 我当众喊：「周建国，你把五岁的孩子挂车外狗笼，还是人吗？」
- 家族群炸锅
- **章尾**：穿制服的哥哥出现在人群里

### 第 6 章｜底牌露头

- 哥哥看录像，脸色铁青；现场控制、带去做笔录
- 许律师电话：证据已固定，人别散
- 周建国变脸：「家庭内部矛盾」；婆婆：「你会毁了我们一家」
- **章尾**：我冷声：「毁你们的，不是我，是你们自己。」

### 第 7 章｜当众亮牌（法律）

- 行车记录仪 + 手机录像播放；危险搭载未成年人
- 女儿送医务室/120：低温 + 惊吓；医生当众质问
- 周建国试图和解：「不就是坐个笼子吗」
- **章尾**：警察：「请配合调查。」

### 第 8 章｜回旋镖（舆论）

- 小姑子「嫂子又作妖」视频被路人转发
- 周建国电话录音「就图个清净」公放
- 家族群聊天记录曝光：「外姓人别惯着」「就俩小时」
- 婆婆改口：「都是建国不懂事，你大人大量」
- **金句**：「五个小时狗笼，你说皮实。现在，你也皮实一个给我看看。」

### 第 9 章｜经济轻补刀

- 我一条微信给爸：撤回担保
- 周建国手机响：领导来电，当场停职
- 围观者议论；金毛副驾 vs 狗笼对比照在群里传
- **章尾**：许律师赶到，离婚协议已在路上

### 第 10 章｜三锤同场（爽度拉满）

- **同一场景连续爆**：法律后果落地 + 舆论发酵 + 停职确认
- 婆婆撒泼抢手机，被路人拦住
- 周建国跪地抓我裤腿
- **章尾**：我甩开：「就俩小时，是吧？机会你在那五个小时里用完了。」

### 第 11 章｜峰值：女儿定音 + 当场离婚

- 周建国试图抱女儿「跟爸爸回家」
- **妞妞**：「你不是爸爸，你把我关笼子里。」——躲到我身后
- 许律师递离婚协议；我：不要周家一分钱，只要抚养权
- 周建国签完，手抖写歪名字
- **章尾**：哥哥开车，我们离开服务区；后视镜里他们一家站在原地

### 第 12 章｜冷离场 + 对比蒙太奇（刺激收束 + 过得很好）

**【上】当天傍晚 · 冷收（约 400 字）**

- 车窗外的服务区越来越远
- 金句：「机会，你在那五个小时里用完了。」
- 妞妞睡着了，小手套还攥着我一根手指

**【下】三个月后 · 对比蒙太奇（约 500–600 字，不另开章）**

| 我（过得很好，≥2 画面） | 周家（对照惨，≥2 画面） |
|--------------------------|--------------------------|
| 带妞妞回娘家，有自己的小床，墙上贴画「妈妈和妞妞的家」 | 周建国被网暴失业，工牌照片还在热搜底下 |
| 我重新做会计，上班打卡；妞妞上幼儿园，复查说恢复不错 | 婆婆在菜市场被指指点点；家族群解散了 |
| 妞妞举着小手套：「妈妈，热。」（对仗开篇「妈妈，冷」） | 小叔子删了「嫂子又作妖」视频，装没事 |
| 春节前夕在娘家包饺子 | 周建国发第 17 条消息，显示未读——已拉黑 |

**收束（推荐主结尾）**

> 妞妞抱着我：「妈妈，我们回家了吗？」  
> 我：「早就在家了。」  
> 手机弹窗闪过周建国的名字，我划掉，给妞妞系围巾。

**对比金句（可选）**：「他们嫌妞妞占地方。现在，整个家都是她的。」

**结尾 LOCK**

- 必须有 **我线 ≥2 个具体「好」** + **周家线 ≥2 个具体「惨」**
- 对比段 ≤600 字，不冲淡第 11 章峰值
- **不合、不原谅、不新欢、不霸总接盘**
- 「妈妈，冷」↔「妈妈，热」对仗闭合

## 十、本篇专属招式

1. **孩子钩子少而狠** — 见第六节台词表  
2. **狗笼三次** — 见第六节  
3. **对比要具体** — 狗 vs 女儿，副驾 vs 狗笼  
4. **必须有外人** — 服务区、路人、家族群  
5. **反杀硬且快** — 第 5 章起动手，第 10–11 章峰值  
6. **结尾双面对照** — 峰值打脸 + 女主过得好（第 12 章下）

## 十二、必避雷区

| 雷区 | 后果 |
|------|------|
| 前 3 章只受气不反击 | 读者骂圣母，划走 |
| 女儿戏份过多变亲子文 | 稀释愤怒 |
| 最后原谅复合 | 评论区炸 |
| 反杀靠霸总救场 | 削弱母亲主体 |
| 经济线写成主锤 | 像《私密照》换皮 |
| 结尾只有惨没有「我过得好」 | 爽感半截，缺对比 |
| 对比段拖成「半年后」长 epilogue | 冲淡第 11 章峰值 |

