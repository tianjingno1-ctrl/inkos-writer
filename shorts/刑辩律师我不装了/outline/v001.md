# 拿我女儿逼我顶罪？刑辩律师我不装了

## 一句话

丈夫伪造合同逼刑辩律师妻子顶罪，拿三岁女儿要挟；妻子取证反杀，当庭送他入狱。

## 人设

| 角色 | 身份 | 功能 |
|------|------|------|
| 程敏 | 刑辩律师，八年退居幕后 | 女主；取证、庭审反杀 |
| 陆衍明 | 明远地产，丈夫 | 反派；顶罪布局、工程命案 |
| 星星 | 三岁半女儿 | 软肋 → 情感核 |
| 吴迪 | 律所实习生，出轨对象 | 从犯；当庭崩溃检举 |
| 秦大志遗孀 | 委托人 | 引线工程坠楼案 |
| 夜班保安 | 证人（不敢出庭） | 门禁+手机证据 |

## 节拍（单篇）

1. 顶罪合同 + 伪造签名 + 用星星逼签 → 女主离家
2. 遗孀递出轨/门禁证据 → 录音内存卡（兔子挂件）
3. 假意妥协 → 舆论抹黑 → 公交站取证 → 劫走星星
4. 手腕红痕 → 宣布鱼死网破
5. 递交经侦/刑侦 → 开庭
6. 庭审证据链：监控、门禁、笔迹鉴定、录音、原始视频
7. 吴迪崩溃 + 结案陈述 + 尾声
