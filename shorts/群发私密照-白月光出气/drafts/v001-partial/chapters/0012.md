# 第12章 浴火重生

风波平息后的第三个月。
君合律所三十六楼的合伙人办公室里，我站在巨大的落地窗前，俯瞰着整座城市的繁华景色。

桌上的咖啡正冒着袅袅热气。
这三个月里，我不仅没有被那场恶劣的私密照风波打倒，反而因为在这场危机中展现出的教科书般的维权手段、严密的证据链逻辑，以及对合规程序的精准掌控，名声大噪。
许多面临类似危机公关和合规风险的企业，纷纷指名道姓要聘请我作为他们的法律顾问。

手机微微震动，是一封新邮件。
发件人是深蓝资本董事会，内容是正式的聘书：邀请我担任深蓝资本大中华区的长期法律合规顾问。
同时，律所管理合伙人也刚刚在全体大会上宣布，我正式晋升为君合律所的全球高级合伙人。

我拿起手机，将顾宸、叶薇，以及顾家所有人的联系方式，从我的手机里彻底删除，拉黑。
那些曾经试图伤害我的泥潭，已经被我远远地甩在了身后。

“咚咚。”
敲门声响起，陆明手里拿着一束鲜艳的郁金香走了进来。
“恭喜你，林合伙人。”陆明笑着将花递给我。
“谢谢师兄，也谢谢你一直以来的支持。”我接过花，闻了闻那淡淡的清香。
“今晚有空吗？深蓝的几个董事想请你吃个饭，顺便聊聊下季度几个项目的合规审查。”陆明说。
“当然有空，工作第一。”我笑了笑。

送走陆明后，我坐回办公桌前，翻开崭新的案卷。
镜子里的我，眼神坚定，神色从容。
面对伤害，哭泣、隐忍和退缩毫无用处。在这个由规则和法律构建的社会里，法律的武器和自身的强大，才是对恶人最完美的解药，也是对自己最体面的救赎。

窗外，朝阳跃出地平线，金色的阳光洒满整间办公室。
我的新人生，才刚刚开始。
