# 类型 · 汉子茶 / 恶女横刀夺爱

> 参考：[ref-小团体里的汉子茶.md](../materials/ref-小团体里的汉子茶.md)

## 适用

- 女频现言、番茄、强情绪、低道德负担（排雷前置）
- 核心不是「甜」，是**拉扯 + 双标 + 读者站女主**

## 结构公式

1. **外壳**：兄弟 / 父子梗 / 汉子茶 — 亲密行为有命名  
2. **内核**：男主只对女主破例；对外「帮你追」对内截胡  
3. **反派/原女主**：可怜但碍事；排外用资源默认分配，不用大吵  
4. **爆发**：现代具体梗（续火花、副驾、误触电话）链式告白
