# 番茄短篇（通用）

适用于：番茄/新媒体短篇、强钩子、身份反转、豪门打脸等。

## 结构偏好

- 章数：通常 8–15 章
- 每章：爽点 + 章末钩子
- 开篇：前 3 章黄金节奏

## 文风

- 短段、对话驱动、手机阅读友好
- 去 AI 味：少「然而」「不禁」「不是…而是…」

## 改稿重点（从 session 沉淀）

- **母版唯一真源**：用户最新定稿 md，不用中间稿/扩写 txt 当基线
- **定点插入**补情节/钩子；禁整章 API 重写（`run-step.mjs` 类）
- **豆包润色**：短语 + 正文，2 章/批；不要润色师/长约束
- **format**：短段混对白（参考汉子茶刷读）；**不要章首【】预告**（开篇钩子 + 场景弹幕【】除外）
- **上架**：`01.`…`10.` 无章标题 + 开篇钩子；简介定稿不改
- **封面**：出图后必验 **600×800** 像素（`fix-cover-fanqie.mjs`）；**女主脸占上半 50–60%** 大特写近景（用户认可，见 retained「封面女主脸大小」）
