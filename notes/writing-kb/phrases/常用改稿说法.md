# 常用改稿说法

> 你对 Agent 说的原话或大意，以及最终是否采用。Agent 在用户说「这句改法以后都用」时追加。

## 格式

```markdown
### YYYY-MM-DD · [书名]

**原话**：…  
**含义**：（Agent 归纳）  
**适用**：润色 / 重写 / 全类型  
```
