| 书评原话 | 对应写法 | 可复用标签 |

| 「鬼点子生成中」（奶牛表情包） | 读者在**玩梗**，说明情绪点要具体、可 meme 化 | `#可玩梗` |

| 「贺予珩好大的尺寸」（更衣室） | 忌妒通过**具体物件/细节**落地，不写抽象「她好酸」 | `#细节忌妒` |

**核心卖点（从片段归纳）**  
1. **汉子茶**：女主以「兄弟/爹/狗儿子」包装亲密，占尽便宜还显得坦荡。  
2. **小团体**：五人私密群 + 对外一致 + 栾可可永远「外人」。  
3. **假帮忙真拆台**：表面帮可可追贺哥，实则挡桃花、抢副驾、抢虾、误导。  
4. **男主双标**：对可可刻薄（「不懂敲门」「不会就别玩」），对枝宁纵容（剥虾、送别墅、吃醋发疯）。  
5. **现代情绪颗粒度**：续火花、误触听到接吻、朋友圈官宣、副驾「她坐过我不坐」。

## 四、可学写法（写你自己的书时用）

### 1. 汉子茶三层

| 层 | 做什么 |
|----|--------|
| 语言 | 爹/儿子/兄弟/蹭 — 把越界行为命名成玩笑 |
| 肢体 | 掐、腿搭、剥虾、同床 — 量大，但**从不自宣恋爱** |
| 对外 | 帮「追他」、叫可可加油 — 道德高地 + 实际截胡 |

### 2. 吃醋要写「具体罪状」

❌ 「我很生气你不在乎我」  
✅ 第 28 章链：**跟别的男人 → 打电话 → 听到接吻 → 误触 → 一天没消息 → 没续火花**

### 3. 排外不用大吵

- 群里聊、当面聊太大声、虾默认是谁的、副驾谁坐过 — **资源与注意力默认分配**。

### 4. 双标要同框

同一章内：对 A 刻薄 + 对 B 剥虾/送别墅 — 读者才爽。

### 5. 忌妒写物件

更衣室：少一条裙子、空衣架、内裤尺码 — 比心理描写省字且上评论。
