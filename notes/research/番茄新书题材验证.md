## 题材 1：快穿 × 对照组/女配 × 及时止损

**推荐等级：P0（冲榜首选）**

### 榜单依据
- 快穿 14 本占榜 56%，绝对主赛道
- 高频词：对照组、女配、万人迷、炮灰、拜金娇气包、白月光、替人渣走正途
- 代表上榜书：《快穿：甜文里的对照组》《快穿：女配稳拿拜金娇气包剧本》《快穿：是炮灰还是娇娇万人迷》《快穿之我替人渣走正途》

### 核心卖点
- 系统/剧本要求女主当对照组、炮灰、女配
- 女主清醒、不卑微、及时止损，不按剧本演
- 原剧情崩坏 → 男主/原女主真香 → 多世界火葬场
- 差异化切口：**对照组不抢男主，只抢人生**（体面退场，各世界独立逆袭）

### 示例书名方向
- 《快穿：白月光对照组，我不干了》
- 《快穿：对照组只负责退场》
- 《快穿：女配剧本我撕了》

### 开篇公式
1. 系统下发任务：当对照组，衬托女主幸福 / 完成炮灰剧情
2. 女主第一世界直接止损（签分手协议 / 撕剧本 / 拒绝修罗场）
3. 原剧情崩坏，原男主追悔
4. 切换下一世界，重复「清醒反杀」爽感

### 与已有能力的匹配
- ✅ 止损型女主（《白月光回国那晚》已验证）
- ✅ 火葬场情绪节奏
- ✅ 细节刀糖
- ⚠️ 需搭建快穿世界观 + 多世界单元结构

## 题材 2：现言豪门 × 替身/白月光 × 群像修罗场

**推荐等级：P1（稳妥，复用已有优势）**

### 榜单依据
- 豪门总裁仅 2 本，竞争小于快穿
- 代表上榜书：《【普女】男友兄弟都对我很好奇》《白月光不想上位》
- 火葬场、追妻仍是女频硬通货

### 核心卖点
- 白月光 / 替身 / 普女人设，但女主不想上位、不抢戏
- 修罗场从 1v1 扩到群像：男友兄弟、合伙人、对手越界
- 女主心死止损 → 独立成长 → 男主（们）追悔火葬场
- 差异化：**普女 + 多人侧目**，章章有张力

### 示例书名方向
- 《白月光不想上位，他们疯了》
- 《普女的我，男友兄弟都越界了》
- 《替身合约到期后，全员火葬场》

### 开篇公式
1. 女主身份明确（白月光 / 替身 / 普女），处境清晰
2. 触发事件：白月光回国 / 合约到期 / 被当众比较
3. 女主冷静止损，不撕逼、不纠缠
4. 男主及群像开始失控，修罗场铺开

### 与已有能力的匹配
- ✅✅ 高度匹配《白月光回国那晚，我终止了替身契约》
- ✅ 火葬场、独立事业线、细节道具（薄荷糖、速写本类）
- ✅ 世界观轻，开书快
- ⚠️ 需避免与已有短篇同质化，群像修罗场是主要差异点

## 题材 3：现言脑洞 × 穿书 × 神豪/马甲

**推荐等级：P2（搏差异化）**

### 榜单依据
- 现言脑洞 5 本，有量但非主赛道
- 代表上榜书：《捞女主播她自带神豪马甲！》《穿成主播文里神豪的恶毒前爱播》《穿成耽美文小受的炮灰妹妹》
- 核心：**身份反差 + 打脸 + 舆论/直播反转**

### 核心卖点
- 穿书成恶毒女配 / 前爱播 / 炮灰身份
- 隐藏神豪 / 平台股东 / 大佬马甲
- 直播、舆论、打脸节奏快，适合番茄短平快
- 差异化：身份双层反转（表面 loser，实际幕后金主）

### 示例书名方向
- 《穿成恶毒前爱播，马甲是平台股东》
- 《捞女主播她自带神豪系统》（榜单已有，需换切口）
- 《穿成炮灰妹妹，我哥们的马甲掉了》

### 开篇公式
1. 穿书身份落地（恶毒前爱播 / 被全网骂）
2. 第一个打脸场景（小号出手 / 马甲露一角）
3. 原书主角团震惊，舆论反转
4. 引出更大冲突（真神豪现身 / 原书剧情偏移）

### 与已有能力的匹配
- ✅ 打脸逆袭、女主独立
- ⚠️ 需强脑洞，易与海量穿书文撞车
- ⚠️ 直播/神豪设定需贴合平台读者口味

|------|----------|------|

## 单次验证检查项

每验证一本，建议至少完成：

- [ ] 书名 + 一句话梗定稿
- [ ] 前三章章纲（每章 1–2 句核心冲突）
- [ ] 开篇 500 字（或 InkOS 完整短篇包）
- [ ] 自检：钩子是否 3 段内立住人设 + 冲突
- [ ] 自检：是否有明确「下一章想看什么」的悬念
- [ ] 记录数据或主观手感，决定升 P0 或搁置
