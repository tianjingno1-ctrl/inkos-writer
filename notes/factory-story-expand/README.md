# factory-story-expand

把火花扩写点**按章节切片**，**一步一步**发给豆包 2.0，每次只改一个点。

## 快速开始

```powershell
cd d:\蜜蜂族\工作台\writer\inkos\notes\factory-story-expand
node run-step.mjs --status
node run-step.mjs --next
```

## 环境

- API Key：`%USERPROFILE%\.config\volcengine\ark-api-key`
- 默认模型：`doubao-seed-2-0-pro-260215`（环境变量 `DOUBAO_MODEL` 可覆盖）
