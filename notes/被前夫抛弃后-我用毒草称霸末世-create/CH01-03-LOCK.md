## 第 1 章 · 毒种（末世日 + 系统绑定）

### 事件链（顺序不可乱）

1. **强钩子切入**：陈家地下仓库/高温洗衣/陈建宇火系压迫——**不要**婚前甜蜜长回忆开场  
2. **末世规则落地**：全球秩序崩坏已发生（断电、通讯中断、进化潮等，择 2–3 项写清即可）  
3. **婚姻闪回**（**50–200 字**）：错嫁、公婆小姑登堂、曾信过他——立刻切回现在  
4. **父亲线预兆**：接应车队/电话失联/爆炸侧写——许振山**不露面**  
5. **婆家变脸**：久无父亲音讯后，克扣食物、当佣人使唤（背景：王老板式世界仍在转，一句即可）  
6. **种植系统绑定**（全文 **40% 前**）→ 仪式感：【叮】→ 扫描 → 匹配 → 绑定成功  
7. **绑定理由**：与「培育/植物/生物活性」挂钩（她曾为父亲温室选种，或偷偷养过一盆活物）  
8. **第一颗种子**：**嗜睡藤**入手（掌心烫/藏指甲缝）→ **当章可不破土**，但须完成绑定 + 种子到手  
9. **章尾**：系统任务弹出 + 陈家门外异动/进化潮二段预兆 + 「下一株种在哪」类钩子

### 系统两层（写作必分）

| 层级 | 是什么 | 谁看得见 |
|------|--------|----------|
| **区域层** | 末世/进化潮公共规则（若设定为全民可见则旁人可反应） | 按世界观统一，勿前后矛盾 |
| **金手指层** | 种植系统 | **仅许蔓** |

### 锁定 UI（绑定段 · 可调数字，结构不可缺）

```
【全球进化潮：第二阶段倒计时 / 或等价区域警告】

【检测到生物培育亲和度】
【扫描进度：……% → 100%】
【匹配度：优】
【种植系统开始绑定……】
【绑定进度：……% → 100%】
【绑定成功】
【宿主：许蔓｜等级：种植师 Lv.1】
【激活格位：1 格（隐蔽：仓库角落/鞋垫夹层等）】
【肥力值：10/10】（或 survival 等价资源名，全书统一）
【新手任务：培育第一株功能性植物（0/1）】
【奖励预览：嗜睡藤种子 ×1（已发放）】
【预览：成熟后可诱发目标嗜睡，持续约 10–30 分钟｜需活体试验】
```

### 本章禁止

- 婚姻回忆 **>200 字**  
- 大段囤货清单  
- 嗜睡藤**公开**放倒陈家（留 ch3）  
- 救援队、诈死、大逃亡  
- **程砚之、许振山正面出场**  
- 苏蔓/裴行/陆峥/花店守家梗  

### 出章状态

系统 Lv.1｜格位 1｜肥力值满｜嗜睡藤种子在手｜**未试验成功**｜陈家压迫持续｜父亲失联预兆已埋

## 第 2 章 · 他们有了异能

### 事件链（顺序不可乱）

1. **第一次/第二次进化潮后果**：陈建宇火系、**陈小姑速度系**（或等价战斗异能）已觉醒  
2. **羞辱可视化**：当众称许蔓「无异能废物」——克扣配给、增加劳作  
3. **许蔓反应**：表面顺从，内心记账——**禁止**当章硬刚  
4. **暗中种藤**：利用格位 1 在监控盲区种下 **嗜睡藤**（耗肥力值，写清数字）  
5. **试验失败或半成品**：如剂量不够/未成熟——**尚无公开打脸**  
6. **章尾**：系统提示「需 48h 成熟」或「需活体试验 1/1」+ 陈小姑进一步刁难钩子

### 本章禁止

- 植物公开放倒全家  
- 许蔓逃跑  
- 程砚之出场  
- 许振山露面  

### 出章状态

嗜睡藤 **生长中**（写清阶段）｜肥力值减少｜羞辱升级｜试验 **0/1 或失败一次**

## 第 3 章 · 第一株见效

### 事件链（顺序不可乱）

1. **小规模生效**：嗜睡藤成熟/花粉/藤蔓碎屑进入目标食物或空气——**可视化**（陈母打盹跌撞 / 小姑失控说漏嘴 / 陈建宇短暂失态，择一）  
2. **许蔓旁观确认**：系统反馈试验成功 + 记录数据（持续时间、目标人数）  
3. **不揭穿**：陈家归因为疲劳/进化潮副作用，**不知**是许蔓  
4. **内心决断**：「能让他们睡过去，就能让他们闭嘴——**能逃，但还不是时候**」（物资/情报/格位不足）  
5. **新钩子**：听见「救援队」「名额」「抛弃」风声之一——**不执行**，只埋  
6. **章尾**：第二株预告（致幻花/情绪花粉灰锁）或陈家新刁难

### 锁定 UI（试验成功段 · 结构不可缺）

```
【试验：嗜睡藤 · 活体 1/1 · 成功】
【效果：目标嗜睡，持续 __ 分钟】
【肥力值：__/__】
【格位：1/1｜嗜睡藤：冷却/可回收】
【新任务：……】
【未解锁：致幻花（需 Lv.2 / 击杀或资源）】
```

### 本章禁止

- **大逃亡 / 诈死 / 救援队正面冲突**（留第 4–25 章）  
- 陈家确认许蔓有异能  
- 第三章恋爱线、程砚之出场  

### 出章状态

**植物已可视化生效 1 次**｜许蔓确认控场可行｜诈死计划未启动｜H002 仍 open（25 章前回收）
