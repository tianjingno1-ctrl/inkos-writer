# 拉扯·暧昧·擦边 写文指南（番茄合规）

**总原则**
- 每世界 **3–5 章至少 1 处** 有效拉扯（有信息推进或好感变化，不是纯贴贴）。
- **单章内宜分 2–3 层递进**：远距暧昧 → 语言/眼神暧昧 → 擦边色气（停在该好感档上限，不越档）。
- **暧昧** = 眼神、台词、共谋、距离感、「只有我们懂」。
- **色气** = 体温、呼吸、触碰边界、视线描过唇/颈/腰又收回、「差一点就……」。
- **色气 = 擦边张力**，不写露骨性行为；必须服从好感—距离阶梯。
- **三男主禁止混用同一套**：傅景深不能顶胯，沈野不能商务腔「该是你的」，陆烬不能草莓糖喂嘴边。

**苏晚统一钓法（各世界可微调）**
- 她：**装软、装不懂、装怕**，内心 OS 清醒吐槽。
- 有效拉扯 = 她 **递一个软钩子** → 男主 **用自己的人设方式接住或推开** → 读者感到「差一点就……」

## 世界1 · 傅景深 —— 禁欲掌权者的「越界成本」

**人设色气内核**：二十八岁掌权人，不近女色是名声；碰她是 **破戒**，每一次肢体升级都要写 **他的克制裂开一道缝**。

| 好感 | 暧昧（信息/眼神） | 色气（体温/呼吸/边界） |
|------|-------------------|------------------------|
| **0–20** | 垂眸扫过又移开 | 扶臂多停半秒；气息扫发顶 |
| **21–40** | 挡身前不解释 | 糖放碟指节多停；视线描锁骨收回 |
| **41–55** | 扣腕「能走？」；推行程 | 蹲身上药呼吸乱一拍 |
| **56–70** | 牵手、客房热牛奶 | 指腹擦脚背，他先松手 |
| **71–85** | 定位表；她亲侧脸他不躲 | 喉结滚、耳尖红、不回应 |
| **86–100** | 公开「我的人」 | 公主抱「别动」；擦泪指节蹭唇角 |

**傅景深专属写法**
- **道具**：草莓/糖/创可贴/定位表 = 绑定与占有，低好感只 **放**，高好感才 **贴肤**。
- **声音**：平时冷短 → 对她低哑、话变少但每句都重。
- **禁止 OOC**：不会撒娇、不会当众失态、不会在 50 分公主抱；色气来自 **压抑**，不是话多调情。

**示例一句（71 档）**
> 她凑过来时他没躲，下颌线绷得发紧，等她嘴唇离开侧脸，才用拇指蹭掉她眼角一点水渍——像擦掉什么不该存在的痕迹，又像留住了什么。

## 世界2 · 沈野 —— 镜头内外的「只许我看」

**人设色气内核**：台上 **力量+湿身+顶胯** 是商品；台下对苏晚 **反而收着**，色气 = **看过太多身体，只对她的软想越界**。

| 好感 | 暧昧（信息/眼神） | 色气（体温/呼吸/边界） |
|------|-------------------|------------------------|
| **0–20** | 命令「站远点」 | 挡身前不碰脸；汗湿领口不让她擦 |
| **21–40** | 扣腕拉离人群 | 视线停腰线一瞬 |
| **41–55** | 拍头顶；顺路接送 | 量战袍指节蹭腰 |
| **56–70** | 牵手、云顶别墅 | 短抱即松，耳根红 |
| **71–85** | 「这段只跳给你看」 | 贴耳；彩排厅只对她软 |
| **86–100** | 「你说了算」 | 镜前圈住，呼吸交缠停不吻 |

**沈野专属写法**
- **场景**：彩排厅、后台、直播间隙、量体间、迈巴赫后座。
- **反差**：直播时对万人擦边 → 下播先洗手再让她递毛巾；**对外骚对内洁**。
- **身体语言**：力量感——托举、圈住、从身后虚护；**不用**傅式商务腔。
- **禁止 OOC**：不会穿三件套开会式表白；不会在低好感喂到嘴边；色气靠 **汗、腰、镜、呼吸**，不靠财富砸人。

**示例一句（62 档）**
> 他刚跳完《野烬》，裸腰上还挂着汗，却把手背给她：「毛巾。」顿了顿，「你自己擦，别碰我。」——可苏晚一靠近，他喉结动了一下，没再往后退。

## 世界3 · 陆烬 —— 拆穿与共谋的「危险温差」

**人设色气内核**：学神校草外壳，**看穿她装**却不戳破；色气 = **冷触+阴影+共谋秘密**，越后期越「只有我们两个人知道」。

| 好感 | 暧昧（信息/眼神） | 色气（体温/呼吸/边界） | 单章可叠 3 层 |
|------|-------------------|------------------------|---------------|
| **0–20** | 「演技不错」；远距旁观 | 冰凉指尖；怀表发烫 | 远观→塞糖→一步外停 |
| **21–40** | 「继续装，我帮你看」 | 俯身不碰；气息扫耳廓 | 共谋→半臂→指节划掌心 |
| **41–55** | 放学「顺路」 | 摘叶蹭耳廓；袖口扫腕 | 并肩→描颈→短护 |
| **56–70** | 「这题不想会？」 | 手覆手写字；短抱即松 | 教题→呼吸近→松手 |
| **71–85** | 怀表「给我」 | 额头贴额试温 | 独宠→贴额→合拢 |
| **86–100** | 公开「我的人」 | 天台圈住；停吻一厘米 | 宣告→圈腰→停 |

**Ch39 范例（13/100）**：远距旁观哭戏 → 共谋「下次演叫上我」 → 托腕放糖、俯身「烫的是它还是你」、视线掠唇角，袖口扫腕即离。

**陆烬专属写法**
- **道具**：橘子糖、怀表残片、Hello Kitty 饭盒（给妹妹的反差）、校服拉链拉到下颌。
- **台词**：拆穿式暧昧——「左脸那滴挂了三秒」「我知道你在演，配合你」。
- **体温**：指尖常凉 → 后期仅对她 **变暖** 是情感拐点。
- **禁止 OOC**：不会阳光傻甜、不会傅式砸资源、不会沈式顶胯；校园场景 **走廊/天台/空教室/晚自习**。

**示例一句（25 档）**
> 「哭完了？」他递过来一颗糖，指尖凉得她一激灵，「没哭完也可以先吃，省得下次左脸还是比右脸慢半拍。」

## 章节植入节奏（建议）

| 世界阶段 | 频率 | 说明 |
|----------|------|------|
| 开世界 1–5 章 | 每 2 章 1 处轻拉扯 | 建立「这人设怎么撩」 |
| 中段 6–15 章 | 每 3 章 1 处 + 每 5 章 1 处稍重擦边 | 跟好感档位升级 |
| 收官前 3 章 | 至少 1 处「差一点」名场面 | 为结算留余韵 |

**与主线并行**：暧昧桥段要 **挂钩任务/打脸/线索**（例如傅景深护短、沈野专场、陆烬怀表），不能 isolated 发糖。

## 润色 / AI 写稿自检

1. 本章好感对应哪一档？肢体是否越档？
2. 本章是否有 **2–3 层**：暧昧 → 色气，还是只有一层贴贴？
3. 这段 **只有这个男主** 会这样做吗？
4. 有没有 **想碰—收回—再碰** 的拉扯？
5. 番茄合规：无 explicit；校园线最多停一厘米、牵手、拥抱。
