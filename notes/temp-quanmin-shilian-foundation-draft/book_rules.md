# book_rules

## 叙事视角与镜头规则

全书采用**第三人称限江晨视角**。镜头贴在江晨身后,读者看到的世界=江晨感知到的世界。他看不见的信息(其他试炼者的面板数据、副本未探索区域的地形、NPC的真实想法),读者也看不见。这种限制性视角是核心爽点的来源:每次结算面板公示时,读者和江晨同步看到"原来我拿了第一",和其他角色同步震惊。

**例外情况**:每卷允许2-3次短暂切换到陆铮视角(单次不超过300字),用于展示他的认知进化过程。切换时机必须是"陆铮看懂某个关键规则"或"陆铮做出重大决策"的瞬间,且必须紧接着切回江晨视角,形成"对手升级→主角应对"的节奏。第三卷第120章有唯一一次切换到沈清视角(她读取到系统管理员记忆的瞬间),用于制造悬念。

**镜头禁忌**:
- 不全知叙述。不出现"与此同时,在另一个副本里"这种上帝视角切换
- 不提前剧透。江晨不知道的信息,叙述者也不能暗示。如果某个NPC是关键节点,叙述不能写"这个看起来普通的村民,实际上..."只能写江晨看到的表象,等他天赋感知到节点再揭晓
- 不内心戏过载。江晨是理性型角色,内心戏主要是分析和计算,不写大段情绪描写。其他角色的内心戏一律通过"表情、动作、对话"外化

## 结算面板的固定仪式感

每个副本的结算是本书的成瘾机制,必须形成固定的叙事节奏:

**第一步:战斗结束的疲惫与忽略**。副本目标完成(守住村落/击杀BOSS/探索到终点),所有试炼者都在清点装备、治疗伤势、讨论战斗表现。这时江晨通常是最不起眼的那个(因为他战斗参与度低、击杀数垫底),会有1-2个炮灰角色冷嘲热讽"F级就是拖后腿""要不是我们,你早死了"。

**第二步:倒计时与紧张**。系统提示"副本结算将在30秒后开始",画面中间出现半透明倒计时。这30秒里镜头会扫过几个关键角色:陆铮擦拭武器、表情自信(他知道自己斩杀第一);某个炮灰还在嘲讽江晨;江晨推一下眼镜、表情平静(他已经心算过自己的权重分)。

**第三步:面板弹出与数据扫描**。系统音效(清脆的"叮"),所有人眼前同时出现结算面板。面板是半透明蓝色、悬浮在视野中央、按总功勋排名从高到低显示所有试炼者的数据。镜头先给陆铮的面板特写:
```
【陆铮】
斩杀功勋:4700
剧情权重功勋:340
总功勋:5040
排名:2
```
陆铮皱眉(他看到自己是第二,但还没往下翻),镜头切到炮灰们开始欢呼"陆哥无敌!""五千功勋太夸张了!"

**第四步:排名反转与打脸**。某个眼尖的人往上翻了一行,看到第一名,声音卡住。镜头给到榜首数据:
```
【江晨】
斩杀功勋:180
剧情权重功勋:6800
总功勋:6980
排名:1
```
镜头扫过所有人的表情:震惊、不解、怀疑、死寂。陆铮盯着面板,沉默三秒,然后抬头看向江晨。炮灰角色开始质疑"系统出错了?""他才杀了几只怪?"

**第五步:江晨的冷静回应**。江晨推眼镜,平静地说一句解释(不是炫耀,是陈述事实):"我守住了村长的房子、完成了三次NPC护送、在BOSS攻击粮仓前提前布置了路障。这些都是主线节点,权重系数12-18倍。"然后转身离开,留下一群人面面相觑。

这个仪式感在前20章每次都完整执行,建立读者的期待;第20章后开始变化(有人学会规则、不再震惊,但依然打不过江晨),保持新鲜感。

## 战斗场面的"非战斗性"

江晨不是战斗主角,所以本书的战斗场面有特殊的写法:**镜头不跟打斗过程,跟江晨的移动和判断**。

**错误示范**:
"陆铮挥动战斧,斧刃带起呼啸的风声,精准斩在魔狼的颈部,鲜血喷涌而出。魔狼哀嚎着倒地,陆铮没有停顿,转身一刀削掉另一只魔狼的头颅..."(这是传统战斗场面,但不适合本书)

**正确示范**:
"陆铮冲进兽群,战斧挥舞的残影在魔狼间撕开缺口。江晨没看他——没必要看,陆铮的输出效率他心里有数。他的注意力在村落东侧,那里有三个村民NPC正在往粮仓搬运物资。天赋冷却结束,淡蓝色光点在视野边缘闪烁,方向指向粮仓地下室。江晨压低身形,绕过主战场,十五秒后抵达地下室入口。一只落单的魔狼堵在门口,他没有硬拼,捡起地上的石块砸向侧面的木桶,木桶滚动的声音吸引魔狼转头,江晨趁机滑进地下室。里面有个瘫坐的老村民,血条只剩30%,头顶有金色叹号标记。江晨蹲下,'能走吗?'老村民抬头,眼神涣散,'粮食...还在上面...'江晨看了眼任务提示,确认触发条件,拽起老村民往出口走。身后传来陆铮的怒吼和魔狼的哀鸣,战斗进入尾声,但那和他无关。"

**写法要点**:
- 陆铮等战斗角色的打斗用"结果描述"带过(斩杀了多少、清空了哪片区域),不细写招式和动作
- 江晨的移动、观察、判断是重点,每个动作都有明确的"规则目的"(这是为了触发XX节点)
- 战斗场面中穿插系统提示(任务进度更新、节点判定成功),让读者清楚江晨在"赚权重分"
- 用环境音效(战斗声、NPC对话、建筑倒塌声)营造战场感,但镜头不离开江晨

## 对话的"信息密度"要求

本书是规则流,对话必须高效传递信息,不能有无效寒暄和情绪拉扯。

**反面示例**:
"江晨,你怎么样?"王倩关心地问。
"还行。"江晨擦了擦额头的汗。
"你刚才好危险,差点被魔狼咬到。"
"嗯,运气好。"
"下次小心点。"
"会的。"
(这段对话零信息量,纯粹占篇幅)

**正面示例**:
"你刚才去了地下室?"王倩问。
"嗯,救了个NPC。"
"触发什么了?"
"粮仓保护任务的前置节点,权重系数15。"
王倩沉默两秒,"你怎么知道地下室有人?"
江晨推了推眼镜,"天赋感知到光点,猜的。"
"猜的准确率这么高?"
"目前100%。"
(每句话都在推进信息:节点内容、江晨的方法、王倩的怀疑)

**对话规则**:
- 每段对话必须完成至少一项功能:交代规则、推进剧情、展示人物认知差、埋设伏笔
- 江晨的对话永远简短、直接、数据化,不说废话
- 其他角色的对话可以有情绪(震惊、质疑、愤怒),但必须快速切入正题
- 避免"复读机对话"(A说一遍,B重复一遍确认),直接跳到下一个信息点

## 权重分的"可见性"设计

剧情权重分是本书核心,但它是抽象的,必须用具体的画面让读者"看见"江晨在赚分。

**设计原则**:每个关键节点触发时,必须有清晰的视觉信号+系统提示。

**示例**:
江晨扶起村长,村长握住他的手,"年轻人,村子能守住,多亏了你。"村长头顶的金色叹号消失,变成绿色对勾。系统提示弹出:
```
【剧情节点:村长生还】已完成
判定:关键NPC存活,村落防御主线完整度+30%
权重系数:×18
获得功勋:1800
```
江晨视野右上角的功勋计数器跳动,数字从2300变成4100。

**要点**:
- NPC头顶的任务标记(金色叹号=可触发,绿色对勾=已完成,红色叉=失败)是最直观的视觉反馈
- 系统提示用代码框格式呈现,保持UI感
- 功勋计数器的数字跳动是即时爽感的来源,每次跳动都是一次小高潮
- 在大型副本中,江晨会手动打开"节点进度表"(半透明窗口,列出当前副本所有已知节点和完成状态),这个画面会在关键章节出现,让读者看到江晨的"作战地图"

## 陆铮视角切换的触发条件

陆铮的短视角不是随便插入的,必须满足以下条件之一:

1. **认知突破时刻**:陆铮第一次看懂某条规则,必须给他的内心戏,展示"战斗天才的大脑如何理解游戏规则"。例如第18章,他盯着结算面板,脑海里闪回江晨在副本中的所有"奇怪行为"(无视战斗、往没有怪物的地方跑、和NPC说话),然后这些碎片拼成完整逻辑链,最后一个镜头是他的眼神从困惑变成冷冽,"原来如此"。

2. **重大决策时刻**:陆铮做出违背本能的选择(比如第30章放弃追杀高价值BOSS、转而守护节点),必须给内心戏,展示"力量型角色的自我克制"有多难。这种时刻不超过全书5次,每次都是他成长的台阶。

3. **与江晨同框的博弈高潮**:第90章排位赛、第180章冠军赛决赛,两人正面对抗时,允许短暂切换到陆铮视角,用他的眼睛看江晨的布局,增强对手的威胁感("他又领先我一步了")。

**单次视角时长**:200-300字,不超过一个自然段。切换格式用空行+分隔符(如"———"),回切时也用分隔符,保持节奏清晰。

## 后台真相的"碎片化释放"

试炼的本质(高维文明的叙事实验、27次轮回、剧本农场)是全书最大的秘密,不能一次性倾倒,必须按节奏释放碎片。

**释放时间表**:
- 第1-30章:0碎片,读者和江晨一样,只看到表面的"游戏化试炼"
- 第31-60章:1-2个"系统异常"彩蛋(乱码、卡顿、NPC说错词),读者可能注意到,也可能当成无关细节
- 第61-90章:3-4个明确的"不对劲"信号(第78章的观察者NPC、第82章的试炼前史书籍),读者开始怀疑
- 第91-120章:碎片开始拼图(系统日志、BOSS的自我意识、记忆重置协议),读者能猜到"试炼不简单",但猜不到具体真相
- 第121-180章:拼图加速(异界入侵的伪装性质、裁决核心入口、前
