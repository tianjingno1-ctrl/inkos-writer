# 末世植物守家 · 建书简报

> 来源：番茄女频专榜扫榜（科幻末世 / 种田 / 游戏体育·求生，各 Top10）+ 用户定稿 v0.3  
> 扫榜结论：60 本样本中 0 本「植物塔防守家」→ 差异化最大；「穿梦想城镇」三榜 0 样本 → 不单独开，留中后期「守家→建城」

## 一句话定位

别人末世囤货、建安全屋；你末世种花，把一块地种成别人进不来的家。

核心情绪不是「打丧尸很爽」，而是：**看着家一点点变安全、变好看、变富饶 —— 安全感 + 成长可见。**

## 标签（专榜有效词）

正文可玩 PVZ / 星露谷梗；**书名与简介不写**「植物大战僵尸」「梦想城镇」。

## 简介前三行（必须给安全感）

1. 末世/求生已降临
2. 系统已绑定，第一株守家植物已种
3. 第一波威胁已被挡在门外

## 金手指：园丁/植灵师系统

任务驱动，非万能商城。

| 层级 | 植物类型 | 功能 | 读者爽点 |
|------|----------|------|----------|
| 初期 | 豌豆藤、刺墙花 | 挡普通丧尸 | 种下去就生效 |
| 中期 | 食人葵、冰冻兰 | 控场、范围伤害 | 组合策略 |
| 后期 | 灵树、结界花 | 净化区域、建城基础 | 从守家到建城 |

**规则（简单、可视化）：**
- 每种植物占格子，耗「生机值/土壤等级」（末世版阳光）
- 植物可升级、可杂交，不能无限种
- 守不住 = 基地掉血 / 作物被毁 —— 有代价才有紧张感

对标：《我在末世经营中餐厅》的经营反馈节奏；《魔女的农场》2×2 土壤→大领地视觉升级。

## 主线骨架

### 第一阶段（1–80 章）：守一个家

末世降临 → 绑系统 → 第一株植物挡第一波 → 扩基地边界 → 收队友/植物伙伴 → 区域小 BOSS

### 第二阶段（80 章+）：建一座城

威胁降级/区域净化 → 重建城镇基建 → 此时再接「梦想城镇」味，不换核心梗

**事业线**：守家 → 扩基地 → 收人 → 建秩序 → 建城  
**感情线**：无 CP 或 1v1 共建；CP 必须为守家服务（外出找物资 vs 守门口）

## 黄金三章（必出）

| 章 | 必出内容 |
|----|----------|
| 第1章 | 末世/求生规则落地（**不要**囤货清单开场）→ 系统绑定 → 第一颗种子 |
