# 文风定调

## 平台

番茄女生 · 七零年代重生军婚 · 短句 · 快节奏 · 章章有硬结果 · 听书友好

## 写作四定律

> 完整条文：`story/book_rules.md`

1. **禁止全知** — 信息从文中来（对话/动作/物证），不直接告诉读者
2. **配角有出发点** — 做事须图什么、怕什么，禁止工具人
3. **世界要真实** — 年代细节、背景声、人情，让读者能代入
4. **人物有形有神** — 外貌/神态锚点 + 行为见精神，不用情绪标签

## 叙述

- **短句为主**，少用长修饰；事件驱动：谁说了什么 → 谁做了什么 → 结果是什么
- 环境一句带过，不写作文；**极短段 + 段间空行**（见 `.cursor/rules/niandai-junhun-chapter-format.mdc`）
- 对话用中文双引号 `""`

## 对话·动作优先（用户钉 · 写章必遵）

> 本条**优先于**系统泛用的「五感堆砌 / 环境作文 / 气质总结」。描写只当接头，不当主菜。

- **主推进**：对话（谁图什么、逼什么、让什么、试探什么）
- **硬事实**：可见动作（解绳、递盒、攥手、泡脚、拍门）——**事实多过描述**
- **辅推进**：女主限知心理，宜短，须接眼前话或眼前物件；禁止作者讲义、人生总结
- **少写**：外貌气质总结（肃杀/**伟岸**/深邃）、气氛形容词堆、明喻（像……）、旁观对比解说
- 场面骨架：`开口/接话 → 动作或物件半句 → 她心里一顿 → 再开口`
- plan / rewrite-brief 的 Style Emphasis **须带两行**（缺一不可）：
  1. `对话+动作推进；事实>描写；少修饰`
  2. `去AI味戒律：禁作者解说动作；禁玄体感；禁半截词；禁翻译腔拆句；删空词须可见动作顶上或退回，勿硬换新形容词`

## 女主 POV · 禁反复报自家门牌

> 教训：ch18–19 叙述里多次「西排二号的土灶/屋里」，像作者对读者报地址，不像女主过日子。

**林晚棠心里清楚自己住哪，叙述指自家时禁反复用「西排二号」。**

| 场景 | 用 | 不用 |
|---|---|---|
| 在自家屋里/灶房 | 屋里、灶房、土灶、院里、自家 | 「西排二号的…」作叙述主语 |
| 对外办事报门牌 | 可：「西排二号有事，找值班」 | — |
| 旁人口中 | 可：李姐喊「这西排二号…」 | — |
| 方位对比 | 可：东排/西排、西排胡同口 | 勿把「西排二号」当自家代称刷屏 |

写后自检：连读本章，女主叙述指自家是否超过 **0–1 次**门牌；多则改成屋里/灶房/院里。

## 章间接缝（强制 · 防断链）

> 教训：ch11→ch12 曾把「霍回营部、交张嫂带人」压成旁白「霍景深回营部了」，读者接不上。

**谁去哪、谁交给谁，必须有可见的戏，禁止只靠旁白完成跨章位移/交托。**

### 开写前 · 接缝三问

1. 上章末，人停在哪？（门口 / 车上 / 屋里）
2. 下章开场，缺了哪一步可见动作？（告别、交待、转身离开、目送）
3. 这句话能不能改成场景？  
   - ❌「他回营部了」  
   - ✅「他说营部有事 → 当面交张嫂 → 上车走人」

### plan / rewrite-brief 必写一行

```text
接缝必写：<因由一句> → <交托/交接> → <离开或目送>（1–3 个可见动作）
```

示例：`接缝必写：门口说明回营部 → 当面交张嫂带路领钥匙 → 林目送后进院`

### 写完自检

连读 **上章末 20 行 + 本章前 20 行**（像听书连播）：中间有没有「没交代就突然在别处」？有则补半场，勿加长告别。

- 不必每章拖长重逢/告别；**一句因由 + 一个交托 + 一个离开** 通常够用
- 「接章」摘要 ≠ 已写成场面：brief 里写了仍要用场面落笔

## 活世界（配角 / 群像 / 生活场景）

> 完整规范：`.cursor/rules/world-ensemble-living.mdc`

- 世界**不因女主到场才转**：进场前已有拍门声、邮差铃、灶房锅铲、邻居碎语
- 陈桂英、陆建国、王德福、李长根、围观婶子——各有一句**人话**、一个**具体动作**，不当背景板
- 禁止「众人」「大家」「人群」；禁止生活段只写女主一人像空镜

## 对话战（世情核心）

```
陈桂英（道德绑架）：「你男人指不定哪天死战场上，你连个孩子都没生……」
林晚棠（算账反杀）：「武装部的章盖在这儿，你贪的180工分，够蹲几年？」
```

- 婆婆话术可抄结构（烈士属/分家/彩礼），**反杀须落到当年政策或物证**
- 每句对话带**目的**：施压 / 试探 / 留证 / 埋钩

## 章节标题（番茄）

- 短、有冲突或反差；可用人名/身份/打脸词
- 正文首行：`第N章 标题`；文件名：`第00N章_标题.md`
- **定名时机**：正文写完、手改定稿、`audit` 通过后再定章名（写作期可用占位名 `第N章` 或大纲功能名）；**ch1 时间锚类标题除外**（如「嫁人第七天」可在开写前锁定）
- **强制**：用户确认定稿 / `approve` **不得**带着大纲功能名或工序词收工（如「预告」「嚼舌根」「断亲」「立足」）。Agent 报定稿前自检：仍是功能名 → 先给番茄备选，勿等用户催
- **定名步骤**：读本章正文 → 抓章末钩子或最大爽点 → 改 `# 第N章` 行 → 重命名 `chapters/000N_*.md` → 同步 `index.json` · `chapter_summaries.md` · `state/chapter_summaries.json` · `current_focus.md` · 大纲章节标题

## 正文版式

- 章标题下空一行；叙述段落后空一行
- 拟声（拍门、砸瓶、叮铃）可独立成段
- CLI 出稿后先按版式手改，再改内容

## 禁用（去 AI 味）

- 不禁、缓缓、微微、眼底闪过、心中涌起复杂情绪
- **伟岸**（气质总结；改可见动作，如站起/理袖口/灯影落肩）
- 章末禁止总结人生、禁止说教式升华

### 改稿戒律（ch23 钉 · 改中/定稿前必过）

> 戒律是**刹车清单**，不是必须逐条砍光的 KPI。撑险感的残留（如「然而/敏捷」类）可留；去味失败就**退回原句**。

1. **禁作者解说动作含义** — 勿「等于当众认了人」「——」后讲义；护短靠动作顺序（先按肩再抬眼）自明
2. **禁玄体感** — 勿「肩头一沉，脚底下稳住了」等读者看不懂在燃什么的体感谜语；莫名就删
3. **禁半截词** — 勿「不结」「来咬」「脉口」；读者须一眼懂在说什么（如「今天先到这儿，明日再说」）
4. **禁翻译腔拆句** — 勿「今天不当场结。要讲理，明日再说。」；并成一句人话
5. **删空词须顶上或退回** — 气焰/冷光/目光如刀等可删；删完掉紧张/掉懂 → 用可见反应（嗓门矮、手顿住）顶；顶不好就退回，**勿硬换第三套形容词**
6. **去 AI 味 ≠ 削平语气和险感** — 成语/旁白若正在撑险感，可列入可接受残留，不强求清
7. **对白去味禁砍秃** — 删修饰可以，**禁**把角色原意砍成半截口号（反例：「避一时没用。得去听全。」）；读者须仍读得出立场、动机、利害
8. **范围** — 只修 P0 时勿顺手清 AI 味（见改正文范围铁律）
9. **宁改写，勿砍空（ch27 钉）** — 二轮去味若句里还有信息/语气：优先**改成更好读的口语**，禁止为去味把意思删光；确属空词才按第 5 条删并顶上或退回
10. **禁硬凑对仗组词（ch26 钉）** — 勿用「A还X，B却Y」「前门走通/考场走通」一类半截对仗顶标签；宁留原句微修，或改成一句人话
11. **标点也扫（ch27 钉）** — 叙述里 `——` 夹注 → 优先拆成两句（句号）；讲义冒号 `：` 后罗列 → 改句号另起；对白/拟声/便条加引号可留

**写前**：`rewrite-brief` Style Emphasis 须带上文「去AI味戒律」那一行（与「对话+动作」并列）。  
**改中检查顺序**：①解说？②半截词？③翻译腔？④玄体感？⑤删完还懂燃点吗？⑥对白原意还在吗？⑦改写是否更顺、有无新对仗？  
**去味后必复读（定稿前多一步 · ch25 钉）**：整章连读一遍，专查「砍秃丢意」——对白是否还能读出原来立场/利害；叙述删段后上下是否断气；半截口号、电报腔短句是否把意思削没；**改写句**是否别扭。发现问题 → **退回原句或补回原意/再微修**，禁止再换一套更短的空话。  
**定稿前 5 分钟只扫**：破折号夹注/点评 · 讲义冒号 · 气焰/冷光类 · 汇报腔/大帽子讲义 · 关键军令是否人话、对象是否清楚 · **对白砍秃**。

## 禁翻译腔（软约束 · 对齐 ch9–11）

> 写章 Style Emphasis 默认：**极短段 + 活世界侧写 + 四定律**，勿再叠「一段必须只写一种感官」等硬剁句规矩。

- 少写欧化：「事实上」「关于……」「作为一名……」「进行……」「存在着」
- 对白口语，少因为/所以/然而；能写动作就少写「感到/觉得」
- 同段可写相关细节（风+冷、霉味+灯光），但忌形容词堆砌与双描口号（如「凉，也实」式对仗）
- AI 高频词（忽然、竟然、仿佛、微微、猛地）少用；成问题就删或改口语，不必机械「只删不换」

# 写作方法论参考（完整版）

以下方法论是写作质量的完整参考。写作时应内化这些原则。

## 一、去AI味：正反例对照

### 情绪描写
| 反例（AI味） | 正例（人味） | 要点 |
|---|---|---|
| 他感到非常愤怒。 | 他捏碎了手中的茶杯，滚烫的茶水流过指缝，但他像没感觉一样。 | 用动作外化情绪 |
| 她心里很悲伤，眼泪流了下来。 | 她攥紧手机，指节发白，屏幕上的聊天记录模糊成一片。 | 用身体细节替代直白标签 |
| 他感到一阵恐惧。 | 他后背的汗毛竖了起来，脚底像踩在了冰上。 | 五感传递恐惧 |

### 转折与衔接
| 反例 | 正例 | 要点 |
|---|---|---|
| 虽然他很强，但是他还是输了。 | 他确实强，可对面那个老东西更脏。 | 口语化转折 |
| 然而，事情并没有那么简单。 | 哪有那么便宜的事。 | 角色内心吐槽替代"然而" |
| 因此，他决定采取行动。 | 他站起来，把凳子踢到一边。 | 删掉因果连词，直接写动作 |

### "了"字控制
| 反例 | 正例 |
|---|---|
| 他走了过去，拿了杯子，喝了一口水。 | 他走过去，端起杯子，灌了一口。 |
| 她笑了笑，转身离开了房间。 | 她嘴角一扬，转身出门。 |

## 二、六步走人物心理分析

每个重要角色在关键场景中的行为，必须经过以下六步推导：

1. **当前处境**：角色此刻面临什么局面？手上有什么牌？
2. **核心动机**：角色最想要什么？最害怕什么？
3. **信息边界**：角色知道什么？不知道什么？对局势有什么误判？
4. **性格过滤**：同样的局面，这个角色的性格会怎么反应？
5. **行为选择**：基于以上四点，角色会做出什么选择？
6. **情绪外化**：这个选择伴随什么情绪？用什么身体语言、表情、语气表达？

禁止跳过步骤直接写行为。

## 三、配角设计方法论

- 配角必须有反击，有自己的算盘。主角的强大在于压服聪明人，而不是碾压傻子。
- 每个配角的行为动机必须与主线产生关联。
- 核心标签 + 反差细节 = 活人（李长根欲言又止、邮差按铃不等人、陈桂英躲柴火垛看笑话）。
- 通过事件立人设，禁止通过外貌和形容词堆砌。
- 不同角色的说话方式必须有辨识度。
- 群戏中不写"众人齐声惊呼"，挑 2–3 个角色写具体反应。
- **活世界**：生活场景与群像须遵守 `world-ensemble-living.mdc`——世界先转，主角后入；主角不在场时事情也在发生（侧写即可）。

## 四、代入感六大支柱

1. **基础信息交代**：一句话能交代身份、性格、地位——"小爷我乃镇南府世子林峰"
2. **具体化/可视化**：描写具体到读者脑海能浮现——"搪瓷缸白汽直冒""冰镇汽水嘶嘶响"
3. **熟悉感**：接地气的场景自带代入感——"高考后小树林的分手""医院走廊的消毒水味"
4. **共鸣**：主角的困境必须有普遍性——被欺压、不公待遇、被低估
5. **欲望驱动**：
   - 基础欲望（被动）：不劳而获、高人一等、扬眉吐气
   - 主动欲望（期待感）：作者刻意制造的情绪缺口→读者期待释放→释放超过预期
6. **五感描写**：视觉、听觉、嗅觉、触觉、味觉——"潮湿的短袖黏在后背上"

## 五、强情绪升级法（避免流水账）

流水账的修法不是删掉日常，而是给日常加"料"：

1. **加入前因后果**：下班回家→加上"催债电话刚打来"→日常有了紧迫感
2. **情绪递进**：坏事叠坏事——被骂→赶不上公交→手机掉了→直播课结束了→包子噎住了。每层比上一层过分
3. **日常必须为主线服务**：万物皆为"饵"。日常段要么埋伏笔，要么推关系，要么建立反差

## 六、写前自检清单

1. 本章对应卷纲中的哪个节点？是否推进了该节点？
2. 主角此刻利益最大化的选择是什么？
3. 冲突是谁先动手，为什么非做不可？
4. 配角/反派是否有明确诉求和反制？
5. 反派当前掌握了哪些信息？有无信息越界？
6. 章尾是否留了钩子？
7. 有没有流水账？如有，加前因后果或强情绪
8. 本章是否推进了主线目标？
