# ch14 写作指引（院里嚼舌根）

接章权威：`outline/vol1_ch10-15_revised.md` + `ledger_workpoints.md` + ch13 定稿正文

## 接章

ch13 末：霍门口「断了也好」；拒付复电已出；红标一眼未碰。

## 接缝必写

断亲余温（议论/复电余波）→ 出门晒衣 → 旁人先嚼舌根 → 再反杀。  
禁止空降对撕；禁止旁白跳切。

## 功能 / 章型

院内立足 + 半公开小爽。章型：私域打脸。字数目标 **3000**（soft 2500–3400）。

## 必须发生

1. 晒衣场世界先转（绳、水、味、人声）。
2. **主对手新挑剔军嫂** + **吴招娣远侧/半句**（A+B）；话术勿复读 ch13「觉悟孝道」整套。
3. 林反杀有物证或规矩落点；围观 2–3 人；张嫂可帮腔一句。
4. 硬结果：口碑立住（闲话收声等可见后果）。
5. 职级：霍营长。对话中文双引号。

## 禁写

- 霍出场救场；拆红标
- 187；团长；回村
- 吴死死抱包 / 揭包
- 拒付复电全文再念一遍（可点一句余波）

## 文风

番茄极短段；活世界；四定律；章间接缝见 `style_guide.md`。
