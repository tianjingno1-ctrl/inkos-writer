# ch12 写作指引（军属院分房）

接章权威：`outline/vol1_ch10-15_revised.md` + `audit_drift.md`

## 接章

ch11 末：吉普停军属院门口；霍说「分房已打招呼」「张嫂先带认路」；章结在门口「进去吧／到家了」。

## 接缝必写

门口说明营部有事 → 当面交张嫂带认路领钥匙 → 递回红标包裹 → 吉普回营 → 林目送后进院。  
然后世界先转（井台/灶烟），张嫂引路。禁止旁白一句「霍景深回营部了」。

## 功能 / 章型

立足过渡 + 群像 + 分房小打脸。章型：过渡 + 群像 + 小爽。字数目标 **2000**（soft 1700–2300）。

## 必须发生

1. 世界先转：院里晒衣/打水/灶烟/娃跑/军属闲话，张嫂引路带认规矩。
2. 分房刁难：管房干事以「空房留给老家属」拖延/挑剔新人。
3. 小打脸：林亮随军证明（克制，非第四遍公审式亮证）+ 张嫂帮腔 + **霍的字条或口信**落实分房（霍景深本人本章不出场）。
4. 硬结果：林晚棠在军属院拿到固定住处（房号/钥匙/铺盖到位）。
5. 职级统一：霍营长 / 守备营营长。对话用中文双引号。

## 禁写

- 霍再出场「一句话定房」
- 红标「到了部队才能拆」第三遍；红标仍不拆
- 亮证打脸第四遍（可出示证明，主爽靠规矩+字条/口信+张嫂帮腔）
- 187 工分；「团长」职级
- 吴招娣「死死抱包」第三遍（最多远处一笔侧写或完全不出场）
- 分房安顿戏与 ch11 末重复堆叠（勿再写姜糖水/挡车门/护手腕等）
- 揭开吴招娣布包；调查公社内鬼

## 文风

番茄极短段；活世界侧写；口语年代感；四定律（禁全知、配角有动机、年代细节、形神兼备）。
