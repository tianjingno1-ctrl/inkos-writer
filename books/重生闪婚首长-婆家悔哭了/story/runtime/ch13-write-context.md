# ch13 写作指引（电报断亲）

接章权威：`outline/vol1_ch10-15_revised.md` + `audit_drift.md` + `ledger_workpoints.md`

## 接章

ch12 末：西排二号入住；红标靠墙未拆；张嫂约晚上开火；林在灶台收拾。

## 接缝必写

灶台/安顿残留 → 电报进院可见场面 → 井台或晒场当众念电断亲。  
禁止旁白「电报来了」跳切；禁止开篇已无来路地当众撕电报。

## 功能 / 章型

KR2 原生家庭断亲落地 + 当众话术反杀。章型：私域打脸 / 小高潮。字数目标 **2200**（soft 1900–2500）。

## 必须发生

1. 世界先转：院内生活声；可带开火饭余温。
2. 电报到（爹妈/公社要钱）：来路场面化。
3. 当众念电反杀：不回村、钱已清（120≠她欠爹妈；180 已划转；禁 187）。
4. 围观 2–3 人；硬结果：回执或复电/拒付纸面至少一样。
5. 职级口径：霍营长。对话中文双引号。

## 禁写

- 回村办断亲；霍本人出场救场
- 拆红标；187；团长
- 吴招娣「死死抱包」；揭布包
- 第四遍公审亮证（可口语点账）

## 文风

番茄极短段；活世界侧写；口语年代感；四定律；章间接缝见 `style_guide.md`。
