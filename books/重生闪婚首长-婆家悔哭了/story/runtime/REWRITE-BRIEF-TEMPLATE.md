# rewrite-brief 模板（第 N 章）

> 用法：复制为 `chN-rewrite-brief.md` 后填空。章间接缝规则见 `story/style_guide.md`。

重写第 N 章《标题》。对齐 ch9–11 文风，不要电报腔剁句。

接章：<上章末人在哪、还欠什么>

接缝必写：<因由一句> → <交托/交接> → <离开或目送>
（示例：门口说明回营部 → 当面交张嫂带路领钥匙 → 林目送后进院）

硬节拍：
1) …
2) …
3) …

Style Emphasis（与 ch10 一致）：
- 极短段 + 段间空行 + 活世界侧写 + 四定律
- 单段 1–2 句，相关细节可同段；禁止电报腔
- 番茄口语、中文双引号；少忽然竟然仿佛微微猛地；少欧化
