# 第 21 章 memo

## 本章目标
兑现“当面拆红标包裹”的承诺，揭开霍景深的一角身世，达成夫妻间的深度情感触碰

## 关联线索
- H011

## 当前任务
林晚棠与霍景深在煤油灯下，连续完成解绳、拆皮、取出旧军功章与未寄家信的动作，并触碰霍景深参军初衷的往事。

## 读者此刻在等什么
1) 那个贴着红标、被霍景深郑重其事叮嘱“等我回来亲手拆”的包裹里到底藏着什么秘密。
2) 经历了 ch20 的急归重逢后，两人在私密空间里如何从“战友式相处”过渡到“真正的夫妻”。
- 本章动作：完全兑现“拆包裹”的期待；通过包裹里的旧物，部分兑现男主的身世背景，但对信件内容制造新的悬念缺口。

## 该兑现的 / 暂不掀的
- 该兑现：拆开红标包裹 → 看到旧军功章（或立功喜报）、老家合影、母亲手帕裹着的未寄家信。
- 该兑现：情感近半步 → 两人并肩蹲在墙角或炕沿，有一次具体的指尖或身体微触。
- 暂不掀：家信全文 → 仅瞥见抬头或一两个字，信的具体内容留到后期情感大爆发。
- 暂不掀：政审与敌特全貌 → 仅由霍景深口头确认“任务成了”。

## 日常/过渡承担什么任务
- [开篇] → 承接 ch20 的余温，通过霍景深解靴、灯影摇晃营造出只有两个人的静谧气口，把焦点自然引向墙角的红标包裹。
-[中段] → 拆包的过程要写出“仪式感”，通过解开粗麻绳、剥开干燥纸皮的感官描写，体现两人的默契与尊重。
- [结尾] → 情绪收敛，以“信先收着”为由转入休息，为次日的定情礼（ch22）留出情感缓冲空间。

## 关键抉择过三连问
- 主角（林晚棠）本章最关键的一次选择：主动指着墙角问“现在拆吗？”
  - 为什么这么做？因为这是两人约好的，她要确认霍景深对她的信任程度。
  - 符合当前利益吗？符合，通过共享秘密稳固婚姻关系。
  - 符合她的人设吗？符合，她做事利索，不搞扭捏那一套。
- 配角（霍景深）本章最关键的一次选择：将未寄出的家信递给林晚棠看。
  - 为什么这么做？他已从心底认准林晚棠是他的家属，愿意分享最隐秘的伤口。
  - 符合当前利益吗？符合，坦诚是爹系宠溺的底色。
  - 符合他的人设吗？符合，少言但行动上绝对信任。

## 章尾必须发生的改变
- 信息改变：林晚棠得知霍景深参军与母亲去世有关，且家中已无牵挂之人。
- 关系改变：两人之间有了共有的秘密物件，关系从“救人闪婚”向“交心伴侣”实质性推进。## 本章 hook 账
open:
- [new] 未寄出的家信 || 理由：信中提到的“娘”和未寄出的原因，暗示霍景深与老家陆家之外的复杂过往。

advance:
- H011 "红标包裹" → 两人当面拆开，见到军功章与合影（planted → near_payoff）

resolve:
- 无（本章为partial payoff，不彻底 resolve）

defer:
- H012 "雷架焦痕" → 本章不动，理由：霍刚归家，重点在夫妻感情，暂不引向敌特线。
- life-夜霜打苗 → 本章不动，理由：深夜室内戏，不宜切户外。

## 不要做
-禁跳切：严禁直接写“包裹拆开后”，必须有解绳剥皮的过程。
-禁独自拆：必须双人在场，由霍景深主导或两人共同动手。
-禁读信全文：禁止抄录信件内容，只能写林晚棠瞥见的模糊只言片语。- 禁讲大道理：霍景深的身世点到即止，不要写成诉苦大会。
- 禁越界：不要在本章写 ch22 的手表/围巾定情礼，也不要发生接吻等亲密戏。
